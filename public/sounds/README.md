# مجلد أصوات الطبيعة

ضع هنا 5 ملفات mp3 (مدّة 30 ثانية على الأقل، loop آمن):

- `waves.mp3` — صوت أمواج البحر
- `rain.mp3` — صوت المطر
- `wind.mp3` — صوت الريح
- `fire.mp3` — صوت طقطقة النار
- `birds.mp3` — أصوات الطيور

## مصادر مجانية مقترحة (Creative Commons / Public Domain)

- [Freesound.org](https://freesound.org) — اختر CC0
- [Pixabay Music](https://pixabay.com/music/) — مجاني للاستخدام التجاري
- [BBC Sound Effects Library](https://sound-effects.bbcrewind.co.uk/) — للاستخدام التعليمي
