# 🛡️ سند — تطبيق الطوارئ المدرسية

تطبيق طوارئ احترافي للطلاب ذوي الإعاقة في المدارس، مبني بـ **Ionic React + Firebase**.

> هذا المشروع مُصمَّم خصيصاً ليكون **سهل الفهم للطلاب** — كل ملف موثَّق بالعربية، والتقسيم واضح بين الواجهة والمنطق والبيانات.

---

## ✨ الميزات

| الميزة | الوصف |
|---|---|
| 🆘 **زر النجدة** | لمسة واحدة → اهتزاز + إرسال موقع GPS + إشعار ولي الأمر |
| 🗺️ **خريطة الملاجئ** | OpenStreetMap حقيقية مع GPS وحساب المسافات |
| 🧘 **مركز التهدئة** | تنفس 4-7-8 + أصوات طبيعة + رسائل تشجيعية |
| 🎮 **ألعاب التشتيت** | لعبة الثعبان + لعبة الذاكرة |
| 📞 **اتصال طوارئ** | 998 / 997 / 999 + ولي الأمر + منسّق المدرسة |
| 👤 **ملف طبي شامل** | فصيلة الدم، الأدوية، الحساسية، بيانات ولي الأمر |
| 🔔 **إشعارات FCM** | تنبيهات فورية لولي الأمر عند الطوارئ |

## 📁 هيكلة المشروع

```
src/
├── api/         ← كل اتصال بـ Firebase (5 خدمات)
├── components/  ← مكوّنات بمنطق (home, map, calm, contact, profile)
├── ui/          ← مكوّنات بصرية بحتة (Icon, Card, Toggle, …)
├── pages/       ← شاشات الـ Router (5 tabs + auth + games)
├── games/       ← منطق الألعاب منفصل عن الواجهة
├── hooks/       ← React hooks مخصّصة
├── context/     ← مزوّدات الحالة العامة
├── helpers/     ← دوال نقيّة (distance, formatters, validators)
├── types/       ← TypeScript interfaces
└── theme/       ← الألوان والأنماط
```

### لماذا هذا التقسيم؟
- **`ui/`** = صفر منطق، فقط شكل → سهل الاختبار والتعديل
- **`components/`** = منطق + استخدام `hooks/`
- **`api/`** = كل استيراد `firebase/*` يبقى هنا فقط
- **`games/`** = pure functions قابلة للاختبار بدون متصفّح

## 🚀 التشغيل السريع (للويب)

### المتطلبات
- Node.js 20+
- npm 10+
- مشروع Firebase (مجاني)

### الخطوات

```bash
# 1) ثبّت الاعتمادات
cd sanad
npm install
cd functions && npm install && cd ..

# 2) أنشئ ملف .env من المثال
cp .env.example .env
# املأ القيم من Firebase Console → Project Settings → Web app

# 3) شغّل المحاكي + التطبيق (وضع التطوير الكامل)
# في تيرمنال 1:
npm run fb:emulators

# في تيرمنال 2:
npm run dev
# افتح: http://localhost:5173
```

## 🔥 إعداد Firebase

1. أنشئ مشروعاً جديداً في [Firebase Console](https://console.firebase.google.com)
2. فعّل:
   - **Authentication** → طريقة تسجيل: Phone
   - **Firestore Database** → ابدأ في وضع الإنتاج
   - **Cloud Functions** (يتطلّب خطة Blaze المجانية للاستخدام المحدود)
   - **Cloud Messaging** (FCM)
   - **Storage**
3. سجّل تطبيق Web وانسخ القيم إلى `.env`
4. انشر القواعد والوظائف:

```bash
# تحقّق من حسابك
firebase login

# اربط المشروع
firebase use --add

# انشر القواعد + الوظائف + Firestore indexes
firebase deploy
```

## 🤝 للمساهمة (دليل الطلاب)

### قاعدة ذهبية: **مكوّن واحد لكل ملف**

### إضافة شاشة جديدة:
1. أضف الملف في `src/pages/NewPage.tsx`
2. سجّله في `src/routes/AppRoutes.tsx`
3. إذا احتاج بيانات → اكتب hook في `src/hooks/useNewData.ts`
4. إذا احتاج Firebase → أضف خدمة في `src/api/newApi.ts`

### إضافة لعبة جديدة:
1. أنشئ مجلداً في `src/games/yourGame/`
2. اكتب `yourEngine.ts` (منطق نقيّ، لا React)
3. اكتب `useYourGame.ts` (يربط المحرّك بـ React)
4. أضف الصفحة في `src/pages/games/YourGamePage.tsx`
